# fugalang

Системный язык программирования предназначен для разработки низкоуровневого ПО,
такого как операционные системы и драйверы, с упором на максимальную производительность
и прямой контроль ресурсов. Ключевыми принципами современного системного программирования являются читаемый синтаксис,
предсказуемость кода, безопасность памяти.

## Оглавление

## Установка

### Вариант 1: клонирование репозитория

```bash
git clone https://github.com/fugalang/fugu.git
make install
```

## Компилятор fugu

Быстрый компилятор с максимально эффективными механизмами разбора кода и его генерации.
