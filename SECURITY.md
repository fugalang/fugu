# Политика безопасности

## Поддерживаемые версии

Мы поддерживаем безопасность только в последних версиях проекта:

| Version | Supported |
|--------|----------|
| latest | ✅ |
| < 1.0.0 | ❌ |


## Сообщение об уязвимости

Если ты нашёл уязвимость, пожалуйста **НЕ создавай публичный issue**.

Вместо этого сообщи нам напрямую:

- GitHub Security Advisories

Исправления безопасности публикуются в новых версиях:

### При отправке отчёта укажи:

- описание уязвимости
- шаги воспроизведения
- возможности (что она позволяет сделать)
- версию проекта


Спасибо, что помогаешь ❤️
