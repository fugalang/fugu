# Contributing to Fugu

Спасибо, что интересуешься вкладом в проект!

Мы рады любым улучшениям: багфикс, новые фичи, документация или идеи.

---

## Как начать
### Форкни репозиторий
#### Клонируй:

```bash
git clone https://github.com/yourname/fugu.git
```

#### Создай ветку:

```bash
git checkout -b feature/my-feature
```

#### **Перед отправкой PR**

#### Убедись, что:

1. **проект собирается:**

```bash
make build
```

2. **тесты проходят:**

```bash
make test
```

3. **код отформатирован:**

```bash
make fmt
```

## Стиль кода

- Go standard formatting (gofmt)
- простая и читаемая логика
- без лишней сложности


## Pull Request правила

- Один Pull Request = одна задача или одно изменение
- Давай понятное и короткое описание: что изменено и зачем
- Если добавляешь новую фичу - желательно приложить пример использования
- Если фиксишь баг - опиши, как его воспроизвести и как ты его исправил
- Старайся не смешивать рефакторинг и новые фичи в одном PR
- Убедись, что проект собирается и тесты проходят перед отправкой

## Баги

Если нашёл баг:

- опиши шаги воспроизведения
- покажи ожидаемое поведение
- покажи фактическое поведение

## Идеи

Открывай issue с тегом enhancement


**Спасибо за вклад ❤️**
